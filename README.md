## Running the Project

1. `git clone` this repo
2. Run `npm install` in the project directory/folder

### Server-Side

`cd` into `server` directory/folder and run:

```bash
npm run start:dev
```

#### Client-Side

`cd` into `client` directory/folder and run:

```bash
npm start
```
