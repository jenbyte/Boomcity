## Server

Commands must be run from the `server` directory:

### Installation-Server

```bash
npm install
```

### Run-Server

```bash
npm run start:dev
```

### Tests

Just linting:

```bash
npm run lint
```

Run linting, and fix any errors:

```bash
npm run lint:fix
```

Run all tests:

```bash
npm run test
```

---

## Client

Commands must be run from the `client` directory:

### Installation-Client

```bash
npm install
```

### Run-Client

```bash
npm start
```

### Build

```bash
npm run build
```
